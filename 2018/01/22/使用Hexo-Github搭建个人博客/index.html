常见问题：

  • 菜单栏英文变成德文或者乱码等
    NexT主题中文字默认为自动选择，但是会常出现问题，建议固定一种语言。编辑站点配置文件，将language设置成你所需要的语言。例如选择简体中文：

更多语言支持可以参见NexT中文官方中语言支持。
更新后如果还有问题，可以参考hexo的next主题变成 似乎是德语了，我什么也没做，原本是英语，如何解决？ - androllen的回答 - 知乎:

  1. 在hexo 文件夹里主题 next文件夹里的language 文件夹 找到 zh-Hans.yml 重命名 zh-CN.yml
  2. 在hexo 文件夹里的_config.yml 修改 language: zh-CN
  • 更换主题失败
    偶尔会遇到主题更换失败的情况，而且在_config.yml文件中已经修改好了主题名称。可以尝试在themes/your_target_theme目录下执行：
