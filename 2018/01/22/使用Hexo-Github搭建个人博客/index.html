绑定域名

域名申请

个人博客 DexterLei.com 于万网上申请，目前是阿里云旗下子公司。直接申请帐号登录，选择未被注册的域名，下单付款即可。建议有能力尽量购买.com域名，另外新用户可以领取部分优惠。
申请完后需要邮箱验证和实名验证，虽然规定是3-5个工作日，但我申请的时候2个小时内就审核通过了。通过后域名管理控制台如下，显示域名状态为“正常”：

域名解析

选择上图中右边的解析链接，进入域名解析设置，选择添加解析，将下图中的 dexterlei.com 替换为自己的 Github Page 即可。

记录类型选 A 或 CNAME ，A 记录的记录值就是ip地址，Github(官方文档)提供了两个IP地址，192.30.252.153和192.30.252.154，这两个IP地址为 Github 的服务器地址，两个都要填上，解析记录设置*，线路就默认就行，CNAME记录值填你的 Github 博客网址。

完成域名解析的添加后，还需要在本地博客根目录创建 CNAME 文件，不带任何后缀，内容为刚申请的域名，如：

dexterlei.com

保存后更新博客并推送到 Github 即可使用新的域名访问主页。同时，Github Page 的设置中 Custom domain 会自动更新为所设置的新域名：

菜单栏英文变成德文或者乱码等

NexT主题中文字默认为自动选择，但是会常出现问题，建议固定一种语言。编辑站点配置文件，将language设置成你所需要的语言。例如选择简体中文：

更多语言支持可以参见NexT中文官方中语言支持。
更新后如果还有问题，可以参考hexo的next主题变成 似乎是德语了，我什么也没做，原本是英语，如何解决？ - androllen的回答 - 知乎:

  1. 在hexo 文件夹里主题 next文件夹里的language 文件夹 找到 zh-Hans.yml 重命名 zh-CN.yml
  2. 在hexo 文件夹里的_config.yml 修改 language: zh-CN
