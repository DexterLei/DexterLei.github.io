2. conda httperror http none none for url none Anaconda包安装失败

出现这个问题选择删除源中的defaults即可，方法有多种：

  • 在Anaconda Navigator主界面中选择Channels，在界面右边直接删除default即可。
  • 在上图的路径C:\Users\Dexter\.condarc（每个设备不同）中找到.condarc文件，删除红圈中部分后保存即可。
  • 使用命令行，也可以删除，暂时没研究……
