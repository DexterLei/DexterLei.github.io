常见任务

语义分割(Semantic Segmentation)

语义分割就是将像素按照图像中表达语义含义的不同进行分组（Grouping）/分割（Segmentation）。


比如从上图中我们可以看到有“人”骑“自行车”，其中包含两个语义对象，分别是“人”和“自行车”，因此语义分割通过检测并使用不同的颜色按照像素分布将其中的人和自行车分别标注出来。语义分割后的图像一般包含若干种颜色的区域，每一种颜色都代表一类。

目标检测(Object Detection)

目标检测是在图片中对可变数量的目标进行查找和分类。其中重要的区别是“可变”这一部分。

如下图中所示，所有的恐龙这一目标被标注出来：

但需要注意的是，目标检测的检测目标只有两类，即目标和非目标，也就是为什么上图中只标注出了恐龙，而没有标注出其他类别的实物。

目标识别(Object Recognition)

检测并标注出图像中所有的实例，并给出类别信息。

与上述的目标检测不同，目标识别的种类没有限制，通常会标注出多个目标，以供后续处理（比如，对比数据库）。

实例分割(Instance Segmentation)

实例分割，即对图像中的每一个像素点进行分类，同种物体的不同实例也用不同的类标进行标注。

小结

  • detection: find all objects (bounding box)
  • classification: classify objects
  • recognition: whether one is object
  • segmentation: cut all objects (pixel wise)
  • localization: find certain object (bounding box)

Metrics

IoU (Intersection-over-Union) 交除并

定义

IoU，Intersection-over-Union 是目标检测、分割等图像处理任务中常见的评价指标。其计算公式如下：
$$IOU = \frac{DetectionResult \bigcap Ground Truth }{DetectionResult \bigcup Ground Truth}$$
简单说，就是计算真值和预测值之间的重叠率，完全重叠时，$IoU = 1$取最大值。

Dice coefficient

Dice coefficent，Dice（相关）系数，又名 Sørensen coefficent，是一个用以比较两个样本之间相似度的统计量。分别由 Thorvald Sørensen 和 Lee Raymond Dice，于 1948 年和 1945 年独立发明。其别名还包括：Dice similarity coefficient (DSC) 等。
